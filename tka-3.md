## **A. Konsep Utama**
1. Cloud computing merupakan sebuah layanan yang memungkinan user untuk menggunakan resource milik penyedia. Dengan memilih metode yang tepat, user dapat menggunakan layanan cloud lebih efisien.
2. Types of cloud merupakan konsep yang di mana pengguna dapat memilih hak akses sesuai kemauan. Types of cloud meiputi public cloud, private cloud, hybrid cloud, dan community cloud.
3. Perbedaan antara public cloud dan community yaitu public coloud lebih terbuka untuk umum, sedangkan community cloud diperuntukkan untuk tujuan tertentu (lebih tertutup).
4. Agar dapat berjalan dengan stabil, cloud computing memerlukan beberapa persyaratan teknis, yaitu cloud networking models, cloud network, dan cloud topologies.
5. Cloud compting dapat dibuat otomatis, sehingga user tidak perlu langsung berinteraksi dengan provider.
6. Dengan menggunakan Federated Cloud Services, provider dapat terhubung ke provider lain. Misalnya Gopay dan Google: Gopay bekerja sama dengan Google untuk menggunakan layanan Google, seperti Sign-in. 
7. Agar berbagai aplikasi dapat berintegrasi satu sama lain, standarisasi cloud diperlukan di sini.

## **B. Istilah Teknis**
- Istilah: Private Cloud

  Definisi: Layanan cloud yang diperuntukkan untuk satu perusahaan

- Istilah: Public Cloud

  Definisi: Layanan cloud yang dapat diakses oleh publik.

- Istilah: Hybrid Cloud

  Definisi: Gabungan antara private dan public cloud.

- Istilah: Community cloud

  Definisi: Layanan cloud yang dapat digunakan oleh beberrapa organisasi untuk tujuan tertentu.
